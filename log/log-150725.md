```
지호 : 오늘 회의는 팀의 방향성을 정한다.
지호 : 우리는 독특한 구조의 TRPG를 만들고싶다. 기존 책과 다른 게임성을 가져야 한다.
영훈 : RP중심, 전투중심?
지호 : 이야기 중심. 전투 룰이 복잡해도 상관 없다. 선택지 세개만 주어지는 텍스트 기반 게임 이름이?
규호 : 잘 모름.
지호 : 아무튼 그린라이트 올라가서 정식 출판됨.
지호 : 우리가 가장 먼저 추구해야하는것은, 폴라리스, 피아스코 같은 류의 룰북은 이런 룰북은 짧고 특정한 게임플레이만 할 수 있다. 보드게임같다. 이런것을 추구할거냐, 아니면 좀 더 범용적이나 다루는 시대와 배경이 제한된 아포칼립스월드, 던전월드같은 게임. 후자가 게임을 만드는데 더 적합할 것이나, 분량이 매우 많음(책 5권 분량?) 그리고 완전 범용적이고 시대제한도 없는 DND, GURPS, 새비지월드.
용욱 : 우리의 한계는 명확하므로 뒤에껀 날립시다. 우리가 원하는것은 배경 스토리가 있었으면 좋겠고, 독특한 룰을 통해 표현됐으면 좋겠다. 방향성은 폴라리스, 피아스코, 마법서점
영훈 : 게임으로 포팅한다면 1번, TRPG에 머무른다면 2번.
지호 : 폴라리스 게임잼에서 9일만에 만들어졌다고 함.
영훈 : 1번으로 가고싶다.
창우 : 그러면 우리 모두가 세계관을 동일해야 한다.
지호 : 피아스코는 룰은 빡빡하되 세계관은 자유다.
용욱 : 우리는 스토리가 풍부한걸 원하지 않았더냐?
지호 : 룰은 빡빡하되 예제 스토리나, 피아스코처럼 범용룰 새비지월드를 비꼬아서 분위기를 만드는것도. 설정이 여러개면 결국 하나를 골라야할듯.
용욱 : 처음에 하려던게 여럿이 각자 설정을 올려서 합치는 방향을 원했으나... 아무도 안올림...
지호 : 늦어진건 죄송. 우리가 할 수 있는 방향은 어떤 세계관을 선택하는게 먼저. 그리고 룰을 뽑읍시다.
창우 : 세계관을 정한다는걸 빡빡하게 하지 말고 키워드를 정해서 각자 생각해오고 그걸 기반으로 작업.
용욱 : 다 재밌자고 모인건데 각자의 로망 요소들을 잔뜩.
창우 : 스토리는 개인이 원하는 방향으로 하되, 룰은 모두의 스토리를 포괄할 수 있게.
지호 : 그러면 1번은 기각. 2번으로.
용욱 : 세계관 합의가 끝난다면 1번이겠지. 스토리 대통합을 하자면 1번이 난도가 낮을듯.
지호 : 영훈, 빵만 먹지말고 얘기좀. 포팅을 한다면 시나리오는 하나만 있어야할듯.
지호 : 세계관 다 모아서 투표한다도 좋은듯.
창우 : 다들 스토리는 들고 와봐야 1번으로 갈 지 2번으로 갈지 정할 수 있을듯.
용욱, 지호 : 세계관은 안정해져도 시대상 정도는 정해져야 할듯.
성찬 : TRPG에 대해 우리가 잘 모르므로 사전조사, 배경지식, 미니프로젝트, 정기 세션등이 있어야 풍부해질듯.
지호 : 게임을 한번 돌리고 미니룰을 만듭시다. 동의하십니까?
영훈 : 미니룰이라는게 존재할지가 의문.
지호 : 그러면 한번쯤 경험을 해보고, 컨셉 잡는데 도움될 것들을 해보자. 아마추어들은 룰을 어떻게 만드는지 깃헙에 링크 올리겠음. 그걸 읽어보고, 대표 룰 한두개정도 경험을 해봅시다.
지호 : 1번 룰의 대표적인것 : 폴라리스, 피아스코, ... 두개 팀 나눠서 돌려봅시다.
창우 : 두어사긴정도면 끝남.
영훈 : 첫 세션 OR은 절대 끝나지 않음.
지호 : 그러면 날짜 잡읍시다. 다음주 어느날이 편하세요? 금토일 안되고, 수요일? 던월은 일요일에 사람 있을 때 경험해보는것도.
지호 : 우리가 구체적으로 합의한건 - 룰 합의에 앞서 세계관 작성할사람은 세계관 작성을 하고, 룰 하나 잡고 플레이.
용욱 : 올리는 사람이 피드백이 없으면 별로 재미가 없다. 피드백 필.
영훈 : 기술적 시대에 대한 합의는 필요. 중세, 근대, 현대, 근미래. 미래?
지호 : 미래는 반대. 너무 걸출한 룰이 있음.
```
